# @sylph/router

Tiny URL router integration for `@sylph/core` state management.

## Installation

```bash
npm install @sylph/router @sylph/core
# or
yarn add @sylph/router @sylph/core
# or
pnpm add @sylph/router @sylph/core
# or
bun add @sylph/router @sylph/core
```
